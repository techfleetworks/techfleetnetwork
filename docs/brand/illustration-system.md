# Tech Fleet illustration system

Brand Visual Guide §5 defines two illustration families. Both must coexist
on the same surface without clashing.

## Family 1 — Sketch-Fill

Used for UI icons, feature explanations, small diagrams.

- **Outline**: hand-drawn quality, slightly organic. 2–3px stroke.
- **Fill**: 45° parallel hatching, never solid. Hatch lines 1–1.5px.
- **Color**: single flat color per illustration. Tech Fleet Blue
  (`#0056A7`, token `--primary`) or Action Blue (`#1863DC`,
  `--primary-hover`) on white.
- **Figures**: abstract silhouettes, no facial features. Communicate
  relationship through posture and proximity.
- **Connections**: simple curved arrows matching the line quality.

## Family 2 — Engraving

Used for hero sections, large visual statements (planet, world map).

- **Line work**: fine 1–1.5px parallel lines build form via density, not
  shading.
- **Color**: single flat color — the brand mint `#7DD8D0` (token
  `--brand-mint`) on white/transparent.
- **Subject matter**: planets, satellites, habitats, exploration metaphors
  that match the Sage archetype.
- **Scale**: hero-sized only. Do not shrink below ~400px wide; fine lines
  become illegible.

## When to use each

| Surface                                      | Family            |
| -------------------------------------------- | ----------------- |
| UI icons, feature cards, inline diagrams     | Sketch-Fill       |
| Email headers, social posts                  | Sketch-Fill       |
| Landing/hero, large section backgrounds      | Engraving         |
| Empty states (small)                         | Sketch-Fill       |
| Empty states (full-page)                     | Engraving         |
